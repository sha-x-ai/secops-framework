Deploy SOC Framework

First-time setup: create your PAT

1. github.com/settings/personal-access-tokens/new
2. Resource owner: Palo-Cortex
3. Only select repositories: Palo-Cortex/secops-framework
4. Permissions → Repository → Actions: Read and write — both, not just Read (the default is Read; you must change it).
5. Save in 1Password. Paste here each time.
