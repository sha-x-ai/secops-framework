## [1.0.0] - June 2026

### Added

#### SentinelOne Singularity Modeling Rule — New modeling rule

Maps sentinelone SentinelOne Singularity events from `sentinelone_v2_generic_alert_raw` into the XDM schema. Provides 13 XDM field mappings covering event identity, alert metadata, and core artifacts. Creates the dataset placeholder via `_schema.json` so the correlation rule can upload on tenants where sentinelone integration data isn't yet flowing.

#### SOC SentinelOne Threat — New correlation rule

Creates an XSIAM passthrough alert for each SentinelOne Singularity threat, normalized to the SOC Framework endpoint contract.
