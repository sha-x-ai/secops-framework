## [1.0.0] - June 2026

### Added

#### Wiz Findings Modeling Rule — New modeling rule

Maps wiz Wiz Cloud events from `wiz_generic_alert_raw` into the XDM schema. Provides 11 XDM field mappings covering event identity, alert metadata, and core artifacts. Creates the dataset placeholder via `_schema.json` so the correlation rule can upload on tenants where wiz integration data isn't yet flowing.

#### SOC Wiz Finding — New correlation rule

Creates an XSIAM passthrough alert for each Wiz finding, normalized to the SOC Framework Cloud-category contract and routed through NIST IR.
